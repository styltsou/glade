# Known Issues

This document tracks persistent issues or UI polish items that are currently deferred.

## 1. Conflict Resolution
- **Description**: Currently, Git conflicts must be resolved manually if they occur. The application does not yet provide an in-app merge interface.
- **Status**: Potential future feature.

## 2. Double Scrollbars in Raw Mode
- **Description**: When viewing a note in Raw Markdown mode, the `textarea` has its own internal scrollbar which can occasionally conflict with the main container.
- **Status**: UI Polish item.
